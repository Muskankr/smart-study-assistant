# 🤖 Smart Study Assistant

An AI-powered learning assistant designed to simplify complex topics, generate quizzes, and solve doubts instantly.

---

## 🚀 Live Demo
👉 https://smart-study-assistant-app.vercel.app

---

## 📌 Problem Statement

Students often struggle with:
- Understanding complex concepts
- Getting instant doubt resolution
- Practicing effectively with quizzes

---

## 💡 Solution

Smart Study Assistant provides:
- 📘 Simple explanations of any topic  
- 📝 Auto-generated quizzes for practice  
- ❓ Instant doubt solving  

Making learning faster, smarter, and interactive.

---

## ✨ Features

- Explain any topic in simple language  
- Generate topic-based MCQs  
- Ask doubts instantly  
- Clean and user-friendly UI  
- Fast and responsive  

---

## 🛠️ Tech Stack

- **Frontend:** Next.js  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS  
- **Deployment:** Vercel  
- **Version Control:** GitHub  

---

## ⚙️ How It Works

1. User enters a topic/question  
2. Selects feature (Explain / Quiz / Doubt)  
3. System processes input  
4. Displays intelligent response
